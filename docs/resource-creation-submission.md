## Short description

SignalGuard Source-Grounded Review Resource Pack is a public set of reusable examples, fixtures, and review checklists for GenLayer builders testing source-grounded Intelligent Contracts. It packages sample claim cases, adversarial claim cases, public Portal dashboard checks, deterministic calculation fixtures, and milestone evidence into one reviewer-friendly index.

## Why it is distinct

- The Project submission is the implementation package.
- The Research & Analysis submission is the methodology and findings.
- The Benchmark submission verifies calculations and package checks.
- This Resource Creation submission is a reusable artifact set that other builders can copy, adapt, and use for their own source-grounded contract tests.

## Guardrails

- Submit once under the clearest fitting category.

- Do not claim the resource pack guarantees Portal acceptance or points.
