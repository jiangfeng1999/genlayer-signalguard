## Short description

SignalGuard Reviewer Quickstart Documentation is a concise reviewer guide for reproducing the public SignalGuard evidence package. It explains the contract entry point, Studio deployment assumptions, sample claim path, dashboard checks, local verification commands, and the boundaries between project, milestone, research, benchmark, and resource evidence.

## Why it is distinct

- The Educational Content page teaches builders how to design the pattern.
- The Research & Analysis page evaluates the methodology and risks.
- This Documentation page is an operational quickstart for reviewers who need to inspect and reproduce the submitted artifacts quickly.
