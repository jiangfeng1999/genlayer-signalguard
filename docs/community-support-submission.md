## Description

SignalGuard Community Support Triage Guide helps builders route common questions about project evidence, Portal status, wallet or reCAPTCHA boundaries, and reviewer links. It is designed as support documentation for the SignalGuard evidence package rather than official GenLayer support.

## Review boundaries

- This is support documentation, not a claim to operate official GenLayer support.
- Wallet signatures, OAuth, reCAPTCHA, and final Portal submissions remain manual user actions.
- Portal submission still requires the user to complete wallet, reCAPTCHA, and final confirmation.
