## Description

SignalGuard Explorer Lens is a compact reviewer index for the deployed SignalGuard GenLayer Intelligent Contract. It validates the contract address format, links the Studio import path, links the Studio explorer address page, and ties those references back to the public project evidence hub. The goal is to make contract review faster and reproducible without requiring private Portal access.

## Why it fits Explorer

- It organizes the explorer path for a concrete deployed GenLayer contract.
- It provides a reproducible manifest generated by `app/explorer_lens.py`.
- It connects address-level inspection with the public evidence hub and source repository.
- It avoids unverifiable claims about private transactions or hidden state.

## Verification

## Review boundaries

- The page is an explorer companion, not a replacement for the official Studio explorer.
- Reviewers should still open the Studio explorer link directly.
- Portal submission still requires the user to complete wallet, reCAPTCHA, and final confirmation.
