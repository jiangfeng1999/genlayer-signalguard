A process note for packaging reviewable GenLayer Portal submissions without unsafe shortcuts.

Workflow Lessons

  • Keep each category page specific to the category instead of reusing one generic link everywhere.
  • Maintain a queue that separates direct candidates from support-only evidence.
  • Use a verifier that checks files, JSON fixtures, Python compilation, and public page markers.
  • Document manual boundaries for wallet signatures, OAuth, reCAPTCHA, and final Portal submission.
  • Avoid fake engagement, duplicate claims, and unsupported uptime or event statements.
