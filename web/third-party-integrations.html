SignalGuard SourceBridge Integrations

A third-party integration pack for converting public APIs and data sources into source-grounded GenLayer review inputs.

3rd party integrations Observed cap 1,000 BP No API keys

Integration Purpose

SignalGuard SourceBridge packages reusable adapter patterns for external data sources. Each adapter produces a public source URL, a bounded claim, and a review note that can be converted into a review_claim payload for the SignalGuard Intelligent Contract.

The current pack covers price data, weather data, GitHub release metadata, and documentation pages. The design keeps all examples inspectable by reviewers without secrets, OAuth, or wallet access.

Adapters
