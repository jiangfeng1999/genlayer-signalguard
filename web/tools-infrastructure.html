SignalGuard Builder Evidence and Status Toolchain

A public, reproducible toolkit for GenLayer builders to prepare Studio calls, inspect public Portal status, and verify a review package without private account access.

Tools & Infrastructure Public Data Only Updated 2026-07-07

Toolchain Components

Portal Status Checker

Reads public Portal user, leaderboard, contribution type, and GitHub evidence status for a wallet.

scripts/check-genlayer-status.ps1
Snapshot Helper

Saves timestamped local JSON status snapshots and reports point deltas without committing private runtime files.

scripts/save-genlayer-status.ps1
Evidence Verifier

Runs required-file checks, Python compilation, JSON parsing, static page markers, and helper-script tests.

scripts/verify-evidence-package.ps1
Studio Payload Helper

Generates a deterministic `review_claim` payload so reviewers can test SignalGuard in GenLayer Studio quickly.

app/signalguard_cli.py

Limits

  • The tools use public data and local files; they do not prove Portal acceptance.
  • Private submission state still requires a logged-in Portal session.
  • No wallet signing, reCAPTCHA, or hidden account data is required for review.
