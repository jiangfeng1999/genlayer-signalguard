SignalGuard Benchmarks

Reproducible checks for Portal public data access, dashboard calculations, CLI payload generation, and the full evidence package.

Benchmarks Public Data Only Updated 2026-07-07

Scope

These benchmarks check whether reviewers can reproduce the dashboard and SignalGuard helper outputs without private Portal access. They do not test wallet balances, signatures, private submissions, or accepted-point decisions.

Known Limits

  • These benchmarks show reproducibility, not Portal acceptance.
  • Portal API rate limits or temporary errors can affect live checks.
  • The static dashboard includes a bundled snapshot so reviewers still see content if live refresh fails.
