SignalGuard Project Overview

A GenLayer Intelligent Contract project for source-grounded claim review, packaged for the Portal Projects category.

Projects Observed cap 4,000 BP Public evidence

Project Summary

SignalGuard turns a public claim plus a source URL into a consensus-backed verdict. The Intelligent Contract fetches the cited source, asks GenLayer validators to assess whether the claim is supported, contradicted, or inconclusive, and stores the accepted report for later review.

The project is intentionally small enough for reviewers to inspect quickly, but complete enough to show the core GenLayer value: natural-language reasoning, web context, deterministic state updates, and app-side call preparation.

The verifier checks required files, Python syntax, CLI payload generation, dashboard calculations, JSON fixtures, and static page markers.
