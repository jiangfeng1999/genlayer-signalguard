SignalGuard Project Overview

A deployed GenLayer DApp for consensus-backed review of public claims and cited evidence.

Projects Live StudioNet contract Wallet-connected DApp Source verified

What the project does

A user supplies a claim and an HTTPS source. The deployed Intelligent Contract fetches the source inside GenVM, asks GenLayer validators whether the evidence supports, contradicts, or cannot resolve the claim, and stores the accepted verdict. The frontend uses GenLayerJS for live reads, wallet connection, contract writes, and accepted-receipt tracking.

Current boundaries

  • The deployed version stores the latest accepted review rather than full history.
  • The DApp validates HTTPS and input lengths; stronger URL policy belongs in a new contract deployment.
  • A future Milestone should be submitted only after the base Project is accepted and the upgraded contract is deployed.
